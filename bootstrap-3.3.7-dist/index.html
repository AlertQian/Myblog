龙岩助学走访记

2017年4月15日、16日，我们一行共6名志愿者组成的小队，走访了龙岩市的几所小学和一些贫困学生家庭。此次活动目的：一是助力小葵课堂的建设。之前厦门麦田团队与福建师范大学合作，为了弥补乡村学校英语教学的不足，我们共同组织了英语网络课堂项目——小葵课堂，通过网络视屏的方式给孩子们上英语课；二是回访一些小学的麦田图书室……
